# Scripts

This directory contains various scripts used for development, build processes, and localization.

## Structure

- **`local/`**: Contains scripts that are not invoked automatically by any CI/CD pipeline or build process. These are intended for manual use by developers.
- **`utils/`**: Contains utility functions and shared logic that are not meant to be executed directly.
- **`l10n/`**: Contains scripts related to localization (l10n).
  - **`l10n/utils/`**: Shared utilities specific to localization tasks.
